common
------------------------------------------
1. 本项目用于发布maven中央仓库的项目做统一管理和配置。  
2. 本项目采用Maven多模块开发方式，模块介绍如下：
    * common-core       ：公共模块
    * common-exception  ：异常处理模块
    * common-leaned     ：学习模块
    * common-mp         ：MyBatis-Plus
    * common-security   ：权限模块
    * common-swagger    ：swagger接口模块
   
3. 项目介绍
   * 后端主体框架： Springboot 2
   * 持久层： MyBatis-Plus
   * 使用java版本： jdk 11+
   
4. 运行条件
   * 每个模块的运行条件详见模块中的 README.md
